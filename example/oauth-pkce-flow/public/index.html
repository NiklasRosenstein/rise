            How PKCE works:

            1. Generate code_verifier (random 128 chars)

            2. Hash it with SHA-256 → code_challenge

            3. Send code_challenge to authorization endpoint

            4. After auth, exchange code + code_verifier for tokens

            5. Rise validates verifier matches challenge
