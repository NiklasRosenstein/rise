# User Guide

The User Guide focuses on day-to-day usage of Rise: creating projects, authenticating, deploying applications, and troubleshooting common issues.

Use this section if you are:

- Deploying and operating applications with Rise
- Managing access and credentials for app workloads
- Working with images, registries, and deployment lifecycle

For platform setup and infrastructure-level operations, see the Operator Guide.  
For contributing to Rise itself, see the Developer Guide.
